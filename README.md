# Remains

A 2D platformer game built with Godot Engine 4.3.

## Features

- Side-scrolling platformer gameplay
- Shooting mechanics
- Multiple levels including Green Zone, Industrial Zone, and Exclusion Zone
- Dialogue system using Dialogic addon
- Double jump ability
- Various enemy types

## Controls

- **A/D** - Move left/right
- **Space** - Jump
- **Shift** - Shoot (when gun is acquired)

## Project Structure

```
├── assets/          # Game assets (characters, sprites, etc.)
├── dialogues/       # Dialogue timeline files
├── levels/          # Level scene files
├── scenes/          # Reusable scene components
└── scripts/         # GDScript game logic
```

## Requirements

- Godot Engine 4.3

## Getting Started

1. Clone this repository
2. Open the project in Godot Engine 4.3
3. Press F5 to run the game (or run `levels/main.tscn`)
